How does javaScript work?

JavaScript is a single-threaded interpreted programming language. At first a javascript file is sent to the javascript engine by the browser. After that, the engine parse the code.Then the code is structured by AST which is a javascript object that contains a tree representation of the source code. Then the code is converted to machine code. Finally the machine executes the code. This is how Javascript works.

What does JS event loop do?

Event loop is a model in javascript. It collects and process the events to execute the code. It monitors calls stack as well as the call back queue. When the call stack is empty, the event loop takes the first event from the queue and pushes it to the call stack. The call stack runs the code. Each event is a function.
